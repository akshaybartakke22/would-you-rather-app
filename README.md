## Getting Started to the Project by: 

### Basic Setup

```
$git clone https://github.com/akshaybartakke22/would-you-rather.git
$cd would-you-rather
```

### To Install all the packages

```
$npm install
```

### To run react-book-app application

```
$ npm start
```
